🔭 These days I'm building internal web platforms for a large Malaysian property & construction group — full-stack work that spans Django and FastAPI REST APIs, polished SvelteKit and Next.js frontends, and GCP Cloud Run underneath it all.

My mindset is more product engineer than code-writer: what matters is the satisfying outcome users actually receive. In practice, that means treating UI/UX as part of the craft.

A few things I've shipped (private / enterprise):

- **HR performance platform** — Django REST backend (150+ endpoints) + a SvelteKit frontend (70+ routes, role-based UIs), with Azure AD SSO, JWT auth, and GCP Cloud Tasks driving scheduled sync and notification jobs.
- **Event RSVP platform** — Next.js + PostgreSQL/Prisma, load-tested to ~12k RSVPs/hour on auto-scaling Cloud Run, with a Pub/Sub-fanned email + WhatsApp confirmation pipeline. Designed with a modern glassmorphism UI.
- **Incident-management system** — decoupled Next.js frontend and Django API with Azure AD SSO, Celery/Redis background tasks, and server-side PDF report generation.

Outside the core stack, I tinker with generative AI — image generation with Stable Diffusion and voice synthesis with So-VITS-SVC. Agentic workflows have also become a growing part of my life, both in and out of work. Research-wise, I've co-contributed to published ML research: [HeatGen](https://github.com/Shanahan-Suresh/segp-group4a-deep-learning-network), a neural-network tool for predicting heat-absorbing pavement materials, written up in the *Journal of Engineering Science and Technology* ([paper](https://jestec.taylors.edu.my/Vol%2018%20Issue%202%20April%202023/18_2_16.pdf)).
